## How to add in your project
### Settings gradle
```groovy
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
### Dependencies
```groovy
implementation 'com.github.Michael-Karpiy:SimpleBackStack-Lib:Tag'
```
